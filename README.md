# 课程表小程序 (Course Schedule App)

## 项目简介

这是一个基于微信小程序开发的课程表与任务管理应用。它旨在帮助用户更好地规划和管理自己的学习生活。用户可以在此应用中添加和查看课程信息、管理学习任务（包括作业和实验），并获得每日的学习推荐和统计信息。

## 功能模块

### 1. 课程管理 (Course Management)
*   **添加课程:** 用户可以为不同的星期添加课程，设置课程名称、上课时间、地点。
*   **查看课程:** 通过切换星期，可以查看对应日期的课程列表。
*   **数据持久化:** 课程信息保存在本地存储中，关闭小程序后数据依然保留。

### 2. 任务管理 (Task Management)
*   **添加任务:** 用户可以添加任务，包括任务标题、描述、截止日期和类型（作业/实验）。
*   **任务状态:** 任务分为“待完成”、“已完成”和“已过期”三种状态。
*   **任务统计:** 页面顶部会显示“待完成”、“已完成”、“已过期”任务的数量。
*   **数据持久化:** 任务信息保存在本地存储中。

### 3. 推荐学习 (Learning Recommendation)
*   **日期时间显示:** 页面会显示当前日期、星期、时间和鼓励语。
*   **周数计算:** 根据基准日期（2025年12月23日为第17周）自动计算当前周数。
*   **个性化推荐:** 根据星期几显示当日的学习计划建议。

### 4. 个人中心 (Profile)
*   **学习统计:** 显示课程总数、任务总数和已完成任务数。
*   **数据整合:** 统计信息来源于本地存储的课程和任务数据。

## 技术栈

*   **前端框架:** 微信小程序原生框架 (WXML, WXSS, JavaScript)
*   **数据存储:** 微信小程序本地存储 (wx.setStorageSync, wx.getStorageSync)
*   **UI 框架:** 微信小程序内置组件 (如 view, text, button, scroll-view 等)
*   **开发工具:** 微信开发者工具

## 快速开始

1.  **下载安装:** 在手机上安装 [微信开发者工具](https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html)。
2.  **打开项目:** 启动微信开发者工具，选择“打开项目”，然后选择本项目的根目录。
3.  **运行项目:** 点击工具栏的“编译”按钮，即可在模拟器中运行小程序。
4.  **体验功能:** 通过点击底部导航栏或页面内按钮，体验各项功能。
